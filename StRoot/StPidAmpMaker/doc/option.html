Fit options:


"B"------fit dedx~beta*gamma. This will cause one set of fitted band parameters for all particle types. A file "ChannelCollectionName+BGBands.root" will be produced.
"A"------fit amplitude. A file "ChannelCollectionName+Amp.root" will be produced.
"R"------fit resolution.
"T"------iterate de/dx~beta*gamma fitting twice.

A ChannelCollectionName is sth like"nhitsBin_0_10_20_45_ptBin_0_Inf_dcaBin_0_2_50000_", it depends on what ChannelCollection the user have added into the StPidAmpManager.

Please note:
1. Amplitude fitting or resolution fitting won't make sense if the band does not get fitted first. So if "A" or "R" is selected, "B" will be added automatically.
2. "R" will be added automatically if "A" is selected.
3. Only the "*BGBand.root" file contains the calibration constant.
4. In most cases people just want to produce parameters for PID, the fit option "A" will do the job.
