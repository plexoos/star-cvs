Please find the jobtracker-server RPM : http://madison.star.bnl.gov/xmlrpc/rpm/star-jobtracker-server-0.1-1.noarch.rpm spec file to build your own RPM: http://madison.star.bnl.gov/xmlrpc/rpm/star-jobtracker-server.spec This rpm automatically requires proper MySQL/Apache/php/xmlrpc versions. These can be obtained by using apt package (part of the Scientific Linux default installation) : 
  $> apt-get install mysql httpd php php-mysql php-xmlrpc;
After rpm installation, please follow the 
  /usr/share/doc/star-jobtracker-server-0.1/INSTALL
document to set proper configuration settings. Best Regards, -- Dmitry Arkhipkin Particle Physics Laboratory
