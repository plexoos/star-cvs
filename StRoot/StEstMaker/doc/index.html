StEstMaker

A documentation is attached to each release of this maker and is located in the doc subdirectory in a LaTeX format. The structure of this documentation is inspired from the StarClassLibrary and StEvent documentations. A ps or pdf version can be produced in a similar way by using the associated Makefile.

    cvs co StRoot/StEstMaker
    cd StRoot/StEstMaker/doc/tex
    make (to get StEstMaker.ps)
    make pdf (to get StEstMaker.pdf)
