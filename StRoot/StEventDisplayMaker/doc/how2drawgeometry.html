How to draw the STAR geometry alone :

Sometimes the only thing one wants to see is the GEANT3 definition of the STAR geometry alone. To do that the simple ROOT macro GeomDraw.C has been introduced. It is as simple as type : 
         root4star GeomDraw.C
Some particular version of the geometry can be drawn also.

root4star 'GeomDraw.C(const char *geomDescriptor="complete",Float_t bombFactor=2.0, const char *outRootFile="")' ,

where
  • "geomDescriptor" value is "complete" "by default" and it can be either
    1. STAR geometry version like "year2003"
      (see: http://www.star.bnl.gov/STAR/comp/prod/MCGeometry.html)
      Example: root4star 'GeomDraw.C("year2003")'
    2. The proper ZEBRA fz file.
      Example: root4star 'GeomDraw.C("/star/u/potekhin/gstardata/y2003x_complete.fz ")'
  • "Float_t bombFactor" is the so-called "bombFactor" to get the exploiding view of the detector. To get the normal view the "bombFactor" has to be set to 1.0. It is usually useless to apply bombFactor < 1.0.

One can adjust the view via ROOT Browser or ROOT TCanvas

  1. Select the volume you are interesting in with left mouse button
  2. Bring the ROOT context menu up with the right mouse button click
  3. Select class method to execute and click it with left mouse button
List of the usefull TVolume methods and its parameters:
  1. TVolume::Draw(const char *levels="3") - - define the number of the levels of the geometry hierarchy to be drawn
  2. TVolume::SetVisibility -

    0 - everything visible,
    2 - this invisible, but sons are visible
    1 - this visible but sons
    3 - neither this nor its sons are visible

To get the OpenGL view one has to

  1. Turn Qt ROOT Layer on (see: http://www.rhic.bnl.gov/~fine/EventDisplay)
  2. Select "OpenGL view" from the "View" menu of ROOT TCanvas

To adjust the so-called bombFactor you can invoke the C++ statement from the ROOT command prompt:

gGeometry->SetBomb(1.6); ,

To change the background color call:
gPad->SetFillColor(kWhite); ,

"kWhite" background is advised if you want to print the image to the paper
