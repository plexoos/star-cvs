Graphical User Interface of StEventDisplayMaker consists of:

  • Main window – Event and Detector picture
  • "Event Control Panel" to control the event and optional filter selections
  • The informational widget   "Info" to represent  the"maker" status information
  • Through MainWindow View->OpenGL view one can get the OpenGL view of the MainWindow also
New StEventDisplayMaker has been included into doEvents.C and bfc.C STAR macros
