Graphical User Interface of StEventDisplayMaker consists of:

  • Main" window – Event and Detector picture
  • Two control push-button panels with optional filter(s):
    1. "Pad Control Panel" to control the "Main" window and the chain "event loop"
    2. "Event Control Panel" to control the event and optional filter selections
  • Two informational widgets:
    1. "Info" – "maker" status information
    2. "Selection" – the list of the selected elements
New StEventDisplayMaker has been included into doEvents.C and bfc.C STAR macros
