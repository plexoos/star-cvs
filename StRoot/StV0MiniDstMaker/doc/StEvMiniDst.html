Comments on the StEvMiniDst Class

This class contains all the event-wise information which is needed for the extraction of strange particles in this package. It is reference via a pointer from the StV0MiniDst class and by inhertitance also from the StXiMiniDst class.

Currently, this class is incomplete. The run and event numbers are not yet stored.
