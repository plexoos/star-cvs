STAR Software for Analysis of Anisotropic Flow

R.J. Snellings and A.M. Poskanzer
Nuclear Science Division, Lawrence Berkeley National Laboratory, Berkeley, CA 94720, USA
In the STAR ROOT environment c++ programs have been written to analyze STAR data for anisotropic (directed and elliptic) flow. This document is a roadmap to the available documentation and is kept updated. The software consists of three modules which are called from the STAR software chain and described in the overview presentation.

StFlowMaker is the interface with the STAR DST. It creates the microDST StFlowEvent from StEvent after event and track cuts. This is describedc in the README document. One can also view the header, the implimentation, and the interface.

StFlowTagMaker fills the tag database from StFlowEvent. This is describedc in the README document. One can also view the header, the implimentation, and the interface.

StFlowAnalysisMaker does the flow analysis and fills histograms. This is describedc in the README document. One can also view the header, the implimentation, and the interface.
