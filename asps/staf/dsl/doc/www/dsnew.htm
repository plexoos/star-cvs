dsNewDataset

The dsNewDataset function creates a new dataset.

int dsNewDataset(DS_DATASET_T **pDataset, char *dsetName, size_t dsetDim);

Parameters

pDataset
Address of pointer to array for dataset
dsetName
Name of dataset
dsetDim
Dimension of dataset array

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.

Remarks

One entry in the dataset array is used by the root of the dataset. The number of entries that can be added to the dataset is dsetDim - 1.
