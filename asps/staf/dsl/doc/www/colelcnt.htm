dsColumnElcount

The dsColumnElcount function returns the number of elements in a column of a table. The number of elements in a column is one for a scalar or the product of the dimensions for an array.

int dsColumnElcount(size_t *pCount, DS_DATASET_T *pTable, size_t colNumber);

Parameters

pCount
Address to return count
pTable
Handle for table
colNumber
Column number

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.
