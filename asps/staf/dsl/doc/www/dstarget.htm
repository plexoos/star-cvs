dsTargetTable

The dsTargetTable function constructs a target table for a join or project operation.

int dsTargetTable(DS_DATASET_T **ppTable, char *tableName, char *typeName, DS_DATASET_T *parentOne, DS_DATASET_T *parentTwo, char *aliases, char *projectList);

Parameters

ppTable
Location of table handle
tableName
Name of target table
typeName
Name of row type for target table
pParentOne
Handle for first parent table
pParentTwo
Handle for second parent table
aliases
List of aliases names for parent tables
projectList
List of columns for target table

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.

Remarks

The columns of th target table are specified by projectList which has the format '{' select_column_specifier ',' .... '}'. A select_column_specifier is either a single column_name or table_name '.' column_name column_name. A select_column_specifier specifies a column in ppTable and a column in either pParentOne or pParentTwo.
