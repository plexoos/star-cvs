dsColumnTypeCode

The dsColumnTypeCode function returns the type code of a column in a table.

int dsColumnTypeCode(DS_TYPE_CODE_T *pCode, DS_DATASET_T *pTable, size_t colNumber);

Parameters

pCode
Address for return of type code
pTable
Handle for table
colNumber
Column number

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.

Remarks

See dstype.h for the definition of the DS_TYPE_CODE_T enum values.
