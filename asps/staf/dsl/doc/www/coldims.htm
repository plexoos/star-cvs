dsColumnDimensions

The dsColumnDimensions function returns the dimensions of a column in a table.

int dsColumnDimensions(size_t *dims, DS_DATASET_T *pTable, size_t colNumber);

Parameters

dims
Array for return of column dimensions
pTable
Handle for table
colNumber
Column number

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.

See Also

dsColumnDimCount, dsColumnElcount
