dsColumnName

The dsColumnName function returns a pointer to the name of a column in a table.

int dsColumnName(char **pName, DS_DATASET_T *pTable, size_t colNumber);

Parameters

pName
Location for return of pointer to name
pTable
Handle for table
colNumber
Column number

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.
