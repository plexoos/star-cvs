dsDatasetEntryCount

The dsDatasetEntryCount function returns the number of entries in a datraset.

int dsDatasetEntryCount(size_t *pCount, DS_DATASET_T *pDataset);

Parameters

Count
Location for return of count
pDataset
Handle of dataset

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.
