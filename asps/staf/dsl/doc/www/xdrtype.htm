xdr_dataset_type

The xdr_dataset_type function encodes or decodes type information for a dataset to a XDR sream.

bool_t xdr_dataset_type(XDR *xdrs, DS_DATASET_T **ppDataset, size_t dsetDim);

Parameters

xdrs
Handle for an XDR stream
ppDataset
Location of handle for dataset
dsetDim
Dimension of arrary for dataset

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.

Remarks

This function encodes or decodes type information for the specified dataset. It does not encode or decode table data. Dynamic memory is allocated for decode if ppDataset points to a NULL pointer.
