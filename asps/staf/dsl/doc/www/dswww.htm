Man pages for the dataset library

Function prototypes

Prototypes for DSL functions and data type codes are in dstypes.h.

Functions that return column attributes

dsColumnDimCount - return the dimensionally of a column of array type

dsColumnDimensions - return the dimensions of a column of array type

dsColumnElcount - return the number of elements in a column

dsColumnName - return the name of a column in a table

dsColumnSize - return the size of a column in bytes

dsColumnTypeCode - return the type code for elements of a column

dsColumnTypeName - return the type name for elements of a column

dsFindColumn - return column number for a named column

Functions that return table attributes

dsTableColumnCount - return the number of columns in a table

dsTableDataAddress - return the address of data for a table

dsTableIsType - compares a table row type with a C++ type specifier

dsTableMaxRowCount - return the maximum number of rows for a table

dsTableName - return the name of a table

dsTableRowCount - return the row count for a table

dsTableRowSize - return the size of a row in bytes

dsTableTypeName - return the name for the type of a row

dsTableTypeSpecifier - return a C++ style type specifier for the type of a row

Functions that access table cells

dsCellAddress - return the address of a cell in a table

dsGetCell - copy data from a table cell to a program variable

dsPutCell - copy data from a program variable to a table cell

Functions for navigation of datasets

dsDatasetEntry - return a handle for a dataset entry

dsDatasetEntryCount - return the number of entries in a dataset

dsDatasetMaxEntryCount - return the maximum number of entries for a dataset

dsDatasetName - return a pointer to a dataset name

dsFindEntry - search for a dataset entry by path name

dsIsDataset - checks for dataset handle

dsIsTable - checks for table handle

Functions for creation and modification of datasets

dsAddDataset - add child dataset to a dataset

dsAddTable - add a table to a dataset

dsAllocTables - allocates memory for tables

dsDeleteEntry - delete an entry from a dataset

dsFreeDataset - free a dataset's dynamic memory

dsNewDataset - create a new dataset

dsNewTable - create a new table

dsReallocTable - change the size of a dynamic table

dsSetTableRowCount - set the row count for a table

Functions that perform operations on table data

dsEquijoin - perform an equijoin of two tables

dsProjectTable - project columns from one table into a second table

dsTargetTable - create a target table for a join or project operation

Functions for XDR style I/O

dsMapTable - maps program variables to dataset tables

dsTasProject - projects data for a table from a dataset to a program variable

xdr_dataset - encodes or decodes a dataset to a XDR stream

xdr_dataset_data - encodes or decodes table data for a dataset

xdr_dataset_type - encodes or decodes type information for a dataset

xdrtape_create - create a XDR stream for tape I/O

xdrtape_flush - flush data buffer for a XDR tape stream

Function that return error information

dsErrorCode - returns error codes from dataset library functions

dsPerror - prints error information to stderr

xdrtape_get_error - get error information for the last tape error

xdrtape_perror - print information about the last tape error
