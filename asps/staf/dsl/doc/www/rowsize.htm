dsTableRowSize

The dsTableRowSize function returns the size in bytes of a row in a table.

int dsTableRowSize(size_t *pSize, DS_DATASET_T *pTable);

Parameters

pSize
Location for return of row size
pTable
Handle for table

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.
