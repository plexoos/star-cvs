dsTableTypeName

The dsTableTypeName function returns a pointer to the name of the type of table rows.

int dsTableTypeName(char **pName, DS_DATASET_T *pTable);

Parameters

pNamet
Location for return of pointer to row type name
pTable
Handle for table

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.
