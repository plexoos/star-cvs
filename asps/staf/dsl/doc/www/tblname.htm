dsTableName

The dsTableName function returns a pointer to the name of a table.

int dsTableName(char **pName, DS_DATASET_T *pTable);

Parameters

pName
Location for return pointer to name
pTable
Handle for table

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.
