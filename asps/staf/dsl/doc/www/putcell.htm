dsPutCell

The dsPutCell function copies data from a specified address to a cell of a table.

int dsPutCell(char *address, DS_DATASET_T *pTable, size_t rowNumber , size_t colNumber);

Parameters

address
Source address for cell data
pTable
Handle for table
rowNumber
Row number
colNumber
Column number

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.
