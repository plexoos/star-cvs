dsFindEntry

The dsFindEntry function returns a handle for an entry in a dataset for a specified path.

int dsFindEntry(DS_DATASET_T **ppEntry, DS_DATASET_T *pDataset, char *path);

Parameters

ppEntry
Location for return handle of entry
pDataset
Handle of parent dataset
path
path for for entry

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.
