dsIsDataset

The dsIsDataset function checks for a dataset handle. The result is TRUE if the handle is for a dataset else FALSE.

int dsIsDataset(bool_t *pResult, DS_DATASET_T *handle);

Parameters

pResult
Location to return result of the compare
handle
Handle for table or dataset

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.
