dsAddDataset

The dsAddDataset function adds a child dataset to a parent dataset.

int dsAddDataset(DS_DATASET_T *pDataset, char *dsetName, size_t setDim, DS_DATASET_T *pChild);

Parameters

pDataset
Handle of parent dataset
dsetName
Name of added dataset
setDim
Dimension of added dataset
pChild
Address of array for added dataset

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.
