dsFindColumn

The dsFindColumn function returns the column number for a named column.

int dsFindColumn(size_t *pColNumber, DS_DATASET_T *pTable, char *name);

Parameters

pColNumber
Location for return of column number
pTable
Handle of table
name
name of column

Return Value

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. To get extended error information, call dsErrorCode . To print error information call dsPerror.
