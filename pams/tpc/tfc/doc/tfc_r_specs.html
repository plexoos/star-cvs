TFC_R SPECS

 
Module Functional Description
init_raw_table.c  Initialize the output tables( raw_sec_m, raw_row_in, raw_row_out) for 24 sectors as required.
reformat_new.c read in the minidaq data (only for zero suppressed data), fill raw_sec_m, then find row, pad information, call  reformat_fill_row or reformat_fill_repeat_row to fill the output tables (raw_row_in, raw_pad_in,raw_seq_in,pixel_data_in,raw_row_out, raw_pad_out,raw_seq_out,pixel_data_out)
 reformat_fill_row.c if there is a new raw data, use this programe to find the pixels, bucket, sequences, pad, then fill the output data.
reformat_fill_repeat_row.c Pads in several rows are divided into two read-out borads, when second fill this row information, need to make the padid in raw_pad_in/out in order.
