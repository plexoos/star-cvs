TPG Specifications

 
Module Functional Description
tpg_main.F Main routine.
tgc_clock_frequency.F Returns clock frequency for the FEE.
tgc_cross_and_dip.F Calculates dip and crossing angle based on sector and momentum information.
tgc_drift_velocity.F Returns drift velocity
tgc_drift_volume_length.F Returns length of the drift volume.
tgc_global_to_local.F Based on global TPC coordinates, calculates the sector number and coordinates in the local (sector) system.
tgc_global_to_local_p.F Momentum transformation from global to local coordinate system. 
tgc_global_to_sector.F Returns sector number based on global coordinates.
tgc_local_to_global.F Transformation from local to global coordinate system.
tgc_local_to_global_p.F Momentum transformation from local to global coordinate system.
tgc_local_to_row_pad_time.F Returns row, pad and time bucket based on local coordinates.
tgc_pad_to_x.F Returnc x in local (sector) coordinate system based on pad number.
tgc_row_par.F Returns row parametrisation in the form of: an*x+bn*y=cn.
tgc_sec24_to_sec12.F Converts sector number in 1-24 convention to 1-12 convention.
tgc_sector_row_pad_time_to_local.F Calculates local coordinates based on row bad and time bucket number.
tgc_time_to_z.F Converts time bucket to z in local coordinate system.
tgc_x_to_pad.F Calculates pad number from x in local (sector) coordinate system.
tgc_y_to_row.F Calculates row number from y in local (sector) coordinate system.
tgc_z_to_time.F Calculates time bucket number from z in local (sector) coordinate system.
tgc_z_inner_offset.F Returns time difference between the inner and the outer sector.
