EVR - Usage

The package is driven by a parameter table, evr_evrpar. Here is the meaning of those variables. One new feature is that EVR now fits always dst_track (globtrk) tracks. The user should feel free to experiment with them:

  • vcut: Cut for determining default vertex tracks
  • cutxy: First pass cut in x-y plane for outlier removal. Track should point back to the vertex at most this distance (in cm).
  • cutz: The same as cutxy but for the beam axis. This is usually more relaxed.
  • cut2: Second pass cut for outlier removal. Common value for x-y and z.
  • cut3: Last pass cut value. This is usually a fraction of a cm.
  • ptmin: Minimum pt track need in order to be used in the fit.
