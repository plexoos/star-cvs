EVR : Event Vertex Reconstruction

The evr package will eventually include a set of modules which perform event vertex reconstruction in STAR for A+A, proton+A, p+p and A+A peripheral collisions. Presently there is only one module, evr_am.F, which is intended for high multiplicity Nucleus+Nucleus collisions. This module might also work well for p+A events but should not be used for p+p collisions or peripheral A+A events. The latter will require special event vertex finder/fitter algorithms. The module inputs global tracks from the DST level table globtrk and outputs the event vertex information to the DST level vertex table privert. See also starnote SN0089 for details on the mathematical derivation and the philosophy of the approach.
