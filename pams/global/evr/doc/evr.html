EVR : Event Vertex Reconstruction

The evr package is a set of modules which perform the event vertex reconstruction in STAR. The user can specify what kind of track information should be used in the fitting, e.g. TPC, SVT or GLOBAL tracks. Different approaches and different modules within the package are applicable to different environments (from high luminosity p-p to Au-Au). See also starnote SN0089 for details on the mathematical derivation and the philosophy of the approach.
